<br />

### A unified interface for ANY wallet

```ts
const wallet = new Metamask();
await wallet.connect();
```

### Supported wallets

#### Browser wallets

- Metamask
- Coinbase Wallet
- Wallet Connect v1 and v2
- Smart Wallet (ERC-4337)
- Local Wallet
- Paper Wallet
- Magic.link
- Ethers

#### Backend Wallets

- Private key
- AWS KMs
- AWS Secret Manager
