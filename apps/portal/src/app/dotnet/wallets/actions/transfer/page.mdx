# IThirdwebWallet.Transfer

This method transfers a specified amount of Wei (smallest denomination of Ether) or ERC20s to a specified wallet address on a given chain.

## Usage

```csharp
// Transfer Wei
var receipt = await wallet.Transfer(chainId, toAddress, weiAmount);
// Transfer ERC20
var receipt = await wallet.Transfer(chainId, toAddress, weiAmount, erc20ContractAddress);
```

<Details summary="Parameters">

### wallet (required)

An instance of `IThirdwebWallet`. This represents the wallet from which the Wei is being transferred.

### chainId (required)

The chain ID of the network on which the transfer is to occur.

### toAddress (required)

The address to which the Wei will be transferred.

### weiAmount (required)

The amount of Wei to transfer. It must be 0 or greater.

### erc20ContractAddress (optional)

The contract address of the ERC20 token to transfer. If not provided, the transfer will be in native tokens.

</Details>

<Details summary="Return Value">

### ThirdwebTransactionReceipt

The method returns a `ThirdwebTransactionReceipt` which contains details about the transaction.

</Details>

## Exceptions

- `ArgumentNullException`: Thrown when the `wallet` is null.
- `ArgumentOutOfRangeException`: Thrown when the `chainId` is less than or equal to 0.
- `ArgumentException`: Thrown when the recipient `toAddress` is null or empty.
- `ArgumentOutOfRangeException`: Thrown when the `weiAmount` is less than 0.
