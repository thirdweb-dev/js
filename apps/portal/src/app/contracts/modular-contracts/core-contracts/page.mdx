# Base Contracts

### ERC721

| Name                                                                          | Description                                                                                                                                                                                                                                                                                                                         |
| ----------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [ERC721Base](/contracts/build/base-contracts/erc-721/base)                    | Implementation of the [ERC721A](https://www.erc721a.org/) standard with the capability to mint NFTs.                                                                                                                                                                                                                                |
| [ERC721SignatureMint](/contracts/build/base-contracts/erc-721/signature-mint) | Adds [signature-based minting](/glossary/signature-based-minting) capability to the [ERC721Base](/contracts/build/base-contracts/erc-721/base) contract.                                                                                                                                                                            |
| [ERC721LazyMint](/contracts/build/base-contracts/erc-721/lazy-mint)           | Adds [lazy minting](/contracts/build/extensions/general/LazyMint) capability to the [ERC721Base](/contracts/build/base-contracts/erc-721/base). Batch lazy-mint NFT metadata and have other wallets claim them under conditions you define in a `verifyClaim` function.                                                             |
| [ERC721DelayedReveal](/contracts/build/base-contracts/erc-721/delayed-reveal) | Adds [delayed reveal](/contracts/build/extensions/general/DelayedReveal) to the [ERC721LazyMint](/contracts/build/base-contracts/erc-721/lazy-mint) contract.                                                                                                                                                                       |
| [ERC721Drop](/contracts/build/base-contracts/erc-721/drop)                    | Adds [delayed reveal](/contracts/build/extensions/general/DelayedReveal) and [claim conditions](/contracts/build/extensions/general/DropSinglePhase) to the [ERC721LazyMint](/contracts/build/base-contracts/erc-721/lazy-mint) contract. Batch lazy-mint NFTs, and set up claim conditions to define how your NFTs can be claimed. |

Batch lazy-mint NFT metadata and have other wallets claim them under conditions you define in a `verifyClaim` function.                                                                                                              |

| [ERC1155Drop](/contracts/build/base-contracts/erc-1155/drop)                    | Adds [delayed reveal](/contracts/build/extensions/general/DelayedReveal) and [claim conditions](/contracts/build/extensions/general/DropSinglePhase1155) to the [ERC1155Base](/contracts/build/base-contracts/erc-1155/base) contract. ### ERC20

| Name                                                                                 | Description                                                                                                                                                                                                                                        |
| ------------------------------------------------------------------------------------ | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [ERC20Base](/contracts/build/base-contracts/erc-20/base)                             | Implementation of the [ERC20](/contracts/build/extensions/erc-20/ERC20) standard with support for the [EIP-2612](https://eips.ethereum.org/EIPS/eip-2612) standard.                                                                                |
| [ERC20Vote](/contracts/build/base-contracts/erc-20/vote)                             | Add [vote](https://github.com/thirdweb-dev/contracts/blob/main/contracts/openzeppelin-presets/token/ERC20/extensions/ERC20Votes.sol) capability to the [ERC20Base](/contracts/build/base-contracts/erc-20/base) contract.                          |
| [ERC20SignatureMint](/contracts/build/base-contracts/erc-20/signature-mint)          | Adds [signature-based minting](/glossary/signature-based-minting) capability to the [ERC20Base](/contracts/build/base-contracts/erc-20/base) contract.                                                                                             |
| [ERC20SignatureMintVote](/contracts/build/base-contracts/erc-20/signature-mint-vote) | Adds [vote](https://github.com/thirdweb-dev/contracts/blob/main/contracts/openzeppelin-presets/token/ERC20/extensions/ERC20Votes.sol) capability to the [ERC20SignatureMint](/contracts/build/base-contracts/erc-20/signature-mint) contract.      |
| [ERC20Drop](/contracts/build/base-contracts/erc-20/drop)                             | Adds [claim conditions](/contracts/build/extensions/general/DropSinglePhase) capability to the [ERC20Base](/contracts/build/base-contracts/erc-20/base) contract. Distribute your token for a price under the criteria set up in claim conditions. |
| [ERC20DropVote](/contracts/build/base-contracts/erc-20/drop-vote)                    | Adds [vote](https://github.com/thirdweb-dev/contracts/blob/main/contracts/openzeppelin-presets/token/ERC20/extensions/ERC20Votes.sol) capability to the [ERC20Drop](/contracts/build/base-contracts/erc-20/drop) contract.                         |

### ERC4337

| Name                                                                                      | Description                                                                                                                                                    |
| ----------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [Account](/contracts/build/base-contracts/erc-4337/account)                               | The Account s a non-upgradable, simple smart account that comes with all the basic benefits of account abstraction                                |
| [AccountFactory](/contracts/build/base-contracts/erc-4337/account-factory)                | This factory smart contract is intended to be used to distribute [Accounts](/contracts/build/base-contracts/erc-4337/account) programmatically.                |
| [ManagedAccount](/contracts/build/base-contracts/erc-4337/mananged-account)               | The ManagedAccount is an upgradeable smart contract account written in the [dynamic contract pattern](https://github.com/thirdweb-dev/dynamic-contracts). |
| [ManagedAccountFactory](/contracts/build/base-contracts/erc-4337/managed-account-factory) | This factory smart contract is intended to be used to distribute [ManagedAccounts](/contracts/build/base-contracts/erc-4337/managed-account) programmatically. |
