# OpenEditionMetadataERC721

## Description

The `OpenEditionMetadataERC721` contract is a module that provides metadata management for Open Edition ERC-721 tokens. It includes functionalities for setting and retrieving shared metadata for tokens, making metadata management simple and efficient.

<GithubButtonLink href="https://github.com/thirdweb-dev/modular-contracts/blob/dev/src/module/token/metadata/OpenEditionMetadataERC721.sol" />

## Callback Functions

| Function          | Description                                                  |
|-------------------|--------------------------------------------------------------|
| `onTokenURI`      | Callback function that returns the token URI for ERC-721 tokens. |

## Fallback Functions

| Function            | Description                                                                               |
|---------------------|-------------------------------------------------------------------------------------------|
| `setSharedMetadata` | Sets the shared metadata for the ERC-721 token collection. Requires minter role.           |
