# MintableERC20

## Description

The `MintableERC20` contract is a module that provides minting functionalities for ERC-20 tokens. This module includes features such as minting with and without signatures, and configuration for primary sales recipients.

<GithubButtonLink href="https://github.com/thirdweb-dev/modular-contracts/blob/dev/src/module/token/minting/MintableERC20.sol" />

## Callback Functions

| Function                          | Description                                                                                                     |
|-----------------------------------|-----------------------------------------------------------------------------------------------------------------|
| `beforeMintERC20`                 | Callback function that checks the mint authorization before minting ERC-20 tokens.                              |
| `beforeMintWithSignatureERC20`    | Callback function that verifies a minting request with a signature and processes the mint accordingly.          |

## Fallback Functions

| Function                 | Description                                                              |
|--------------------------|--------------------------------------------------------------------------|
| `getSaleConfig`          | Returns the sale configuration for a token, including the primary sale recipient. |
| `setSaleConfig`          | Sets the sale configuration for a token, including the primary sale recipient. Requires manager role.       |

## Required Interfaces

| Interface  | ID          |
|------------|-------------|
| `ERC-20`   | `0x36372b07`|

## Installation Callback

This module uses an installation callback to set up the initial parameters such as the primary sale recipient.
