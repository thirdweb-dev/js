## ERC721

| Name                                                                               | Description                                                                                |
| ---------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------ |

| [ERC721Mintable](/contracts/build/extensions/erc-721/ERC721Mintable)               | Mint new NFTs into the contract                                                            |
| [ERC721Revealable](/contracts/build/extensions/erc-721/ERC721Revealable)           | NFTs that can be revealed at a later date after they have been minted and claimed          |
| [ERC721SignatureMint](/contracts/build/extensions/erc-721/ERC721SignatureMint)     | Use signature-based minting functionality to mint new NFTs into the contract               |
| [ERC721Staking](/contracts/build/extensions/erc-721/ERC721Staking)                 | Implement ERC721 staking mechanism                                                         |
| [ERC721Supply](/contracts/build/extensions/erc-721/ERC721Supply)                   | View information about the supply of the NFT collection                                    |
