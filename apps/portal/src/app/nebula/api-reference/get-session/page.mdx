#### Get Session

Get details for a specific session.

```bash
GET /session/{session_id}
```

**Example curl:**
```bash
curl -X GET https://nebula-api.thirdweb.com/session/abc123 \
  -H "x-secret-key: YOUR_THIRDWEB_SECRET_KEY"
```

**Response:**
```tsx
{
  "result": {
    "id": "string",
    "title": "string",
    "model_name": "string",
    "is_public": boolean,
    "created_at": "datetime",
    "updated_at": "datetime",
    "messages": []
  }
}
```
