# Account UI Components

Build your own UI for displaying account information with these headless components.

```tsx
import {
  AccountProvider,
  AccountAvatar,
  AccountName,
  AccountBalance,
} from "thirdweb/react";

const App = () => (
<AccountProvider>
  <AccountAvatar />
  <AccountName />
  <AccountBalance />
</AccountProvider>
);
```
