# Web3 Social Identities

Query all web3 social identities for a given address.

## Supported Social Identities

- [Farcaster](https://farcaster.xyz/)
- [Lens Protocol](https://lens.xyz/)
- [ENS](https://ens.domains/)

## Usage with UI Components

When using [`ConnectButton`](/references/typescript/v5/ConnectButton) or [`useWalletDetailsModal`](/references/typescript/v5/useWalletDetailsModal), the user's avatar and username will be automatically populated with their linked social profiles.

## Usage with your own UI

Use the [`useSocialProfiles`](/references/typescript/v5/useSocialProfiles) hook to query all web3 social identities for a given address.
