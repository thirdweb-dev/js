## Usage with UI Components

The easiest way to buy crypto is to use the `BuyWidget` prebuilt UI component.

```tsx
import { BuyWidget } from "thirdweb/react";
import { base } from "thirdweb/chains";

function App() {
  return (
    <BuyWidget
      client={client}
      chain={base}
      amount="0.01"
      title="Get funds"
      // ... theme, onSuccess, onError, etc.
    />
  );
}
```

Check out the [BuyWidget API reference](/references/typescript/v5/BuyWidget) for more information.
