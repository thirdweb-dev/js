# Private Key Wallets

Private Key Wallets are the traditional entrypoint into account management, allowing generation as well as importing an
existing key

## Generate Wallet
Generates a new private key wallet handle.
<Blueprint code={generateWalletNode} className='w-full h-32' />

## Create PrivateKey Wallet
Converts a private key into a wallet handle. FString and FText Variants.
<Blueprint code={createPrivateKeyWalletNode} className='w-full h-52' />

## Get Private Key
Get the private key of a PrivateKeyWallet handle. FString and FText Variants.
<Blueprint code={getPrivateKeyWalletNode} className='w-full h-40' />
