# API Reference

Update configuration, manage backend wallets, and call contracts with a familiar and widely compatible HTTP interface.

<Callout title="Using Typescript?" variant="info">

Use the [Typescript SDK](/engine/references/typescript) for a native, intuitive interface to all of Engine's capabilities.

</Callout>
