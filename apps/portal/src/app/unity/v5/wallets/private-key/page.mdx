# PrivateKeyWallet

`PrivateKeyWallet` is best used as a guest wallet. Should be treated as ephemereal, useful for testing.

### Generating a PrivateKeyWallet

```csharp
var options = new WalletOptions(provider: WalletProvider.PrivateKeyWallet, chainId: 1);
var wallet = await ThirdwebManager.Instance.ConnectWallet(options);
```
