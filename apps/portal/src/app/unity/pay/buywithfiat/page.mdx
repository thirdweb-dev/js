# BuyWithFiat

Trigger a crypto onramping flow. See [GetBuyWithFiatQuote](/unity/pay/getbuywithfiatquote) for more information on how to get an onramp quote.

## Usage

```csharp
using Thirdweb.Pay;

public async void Buy()
{
	string intentId = await ThirdwebManager.Instance.SDK.Pay.BuyWithFiat(_quote);
	ThirdwebDebug.Log($"Intent ID: {intentId}");
}
```

<Details summary="Configuration">

#### BuyWithFiatQuoteResult

See [GetBuyWithFiatQuote](/unity/pay/getbuywithfiatquote) for more information on the `BuyWithFiatQuoteResult` object.

### Return Value

`string`: The intent id that can be used to track the onramp status, as well as link additional Buy With Crypto transactions to this onramp flow.
