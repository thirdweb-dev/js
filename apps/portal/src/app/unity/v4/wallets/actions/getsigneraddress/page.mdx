# GetSignerAddress

Gets the address of the signer associated with the connected wallet.
In most cases, that is the wallet address.
When it comes to Smart Wallets, it is the associated EOA's address.

## Usage

```csharp
string signerAddress = await sdk.Wallet.GetSignerAddress();
```

## Return Value

Returns the `string` address of the signer associated with the connected wallet.
