# RemoveAdmin

For Smart Wallets only, remove an admin from the connected Smart Wallet.

## Usage

```csharp
var txResult = await sdk.Wallet.RemoveAdmin("0xadmin");
```

## Configuration

### admin

The address of the admin to remove.

Must be a `string`.

### Return Value

Returns a `TransactionResult` object.
