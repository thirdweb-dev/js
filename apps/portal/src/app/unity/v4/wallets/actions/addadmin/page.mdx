# AddAdmin

For Smart Wallets only, add an admin to the connected Smart Wallet.

## Usage

```csharp
var txResult = await sdk.Wallet.AddAdmin("0xadmin");
```

## Configuration

### admin

The address of the admin to add.

Must be a `string`.

### Return Value

Returns a `TransactionResult` object.
