# GetAllActiveSigners

For Smart Wallets only, gets all active signers.

Useful for checking if session keys need to be renewed or removed.

Will throw if account is not deployed.

## Usage

```csharp
var activeSigners = await sdk.Wallet.GetAllActiveSigners();
```

## Configuration

## Return Value

Returns a `List<SignerWithPermissions>` object.

```csharp
public struct SignerWithPermissions
{
    public bool? isAdmin;
    public string signer;
    public SignerPermissions permissions;
}

public struct SignerPermissions
{
    public string startDate;
    public string expirationDate;
    public string nativeTokenLimitPerTransaction;
    public List<string> approvedCallTargets;
}
```
